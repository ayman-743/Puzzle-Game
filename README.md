# 🧩 Puzzle Game (Sliding Puzzle)

An interactive sliding puzzle game built using Python and Pygame.
Players can load any image, choose difficulty levels, and solve the puzzle by rearranging shuffled tiles.

---

## 🚀 Features

* 🖼️ Load custom images from your system
* 🎯 Multiple difficulty levels (3x3, 4x4, 5x5)
* 🔀 Smart tile shuffling ensuring solvable puzzles
* 🖱️ Mouse-based tile movement
* 🔊 Sound effects for moves and winning
* 📊 Move counter tracking performance
* 🖼️ Preview of original image during gameplay

---

## 🛠️ Tech Stack

* **Language:** Python
* **Libraries:** Pygame, Tkinter
* **Concepts:** Event-driven programming, 2D graphics, game logic, image processing

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/puzzle-game.git
```

2. Navigate to the project folder:

```
cd puzzle-game
```

3. Install dependencies:

```
pip install pygame
```

4. Run the game:

```
python main.py
```

---

## 🎮 How to Play

1. Launch the game
2. Select an image from your system
3. Choose difficulty level
4. Click on tiles adjacent to the empty space to move them
5. Arrange all tiles correctly to win

---

## 🧠 Game Logic

* The image is divided into a grid of tiles
* One tile space is left empty
* Tiles are shuffled using valid moves
* Player rearranges tiles to restore the original image
